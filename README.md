# Thinkel

Thinkel is a beautiful web application designed to inspire and encourage reflection through daily quotes and personal notes.

Available at: [https://lovro-b.github.io/thinkel/index.html](https://lovro-b.github.io/thinkel/index.html)

## Features

- **Daily Inspiration**: Receive a handpicked quote every day.
- **Quote History**: Browse through previous quotes using the integrated date picker.
- **Favorites**: Save the quotes that resonate most with you for quick access.
- **Personal Notes**: Jot down your thoughts and reflections alongside any quote.
- **Wallpaper Editor**: Create and download beautiful, custom wallpapers with your favorite quotes for your phone or desktop.
- **Multilingual**: Supports both English and Slovenian.
- **Customizable Experience**: Choose between light and dark themes and adjust the font size to your preference.
- **Data Privacy**: All your notes and favorites are stored locally in your browser. You can also export and import backups of your data.

## How to Use

1. **Daily Quote**: The home page displays the quote for today. Use the date icon in the top left to see quotes from other days.
2. **Saving and Notes**: Click the heart icon to add a quote to your favorites. Click the note icon to add your personal thoughts to a quote.
3. **Managing Saved Content**: Go to the "Saved" tab in the bottom navigation to view your favorites and notes. You can search through them using the search bar.
4. **Customization**: Use the "Settings" tab to change the language, theme, and font size.
5. **Creating Wallpapers**: From the Settings menu, open the "Wallpaper Editor". Upload an image, customize the text alignment, size, and font, and download your personalized wallpaper.

## Technologies Used

- HTML5
- CSS3 (with support for Dark/Light themes)
- JavaScript (Vanilla)
- LocalStorage for data persistence
- Canvas API for the Wallpaper Editor
